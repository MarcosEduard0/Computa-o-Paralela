# Compilação e Versão GCC

- **Versão**: 9.3.0
- **Compilação**: gcc -o trabalho_1 trabalho_1.c -lpthread -lm -Wall

Caso prefira, ultilizar o makefile:

```sh
mkdir Trab1
cd Trab1
make
./integral <a> <b> <nº retangulo> <nº de threads>
```
